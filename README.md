# 🎓 Academic & Educational Dashboards

Welcome to the academic dashboards directory. This folder serves as a curated portfolio of business intelligence projects developed for educational research, case studies, and business scenario simulations. 

The primary focus of these projects is to demonstrate end-to-end data pipelines—translating theoretical business problems and raw datasets into structured, executive-ready visual insights.

---

## 🛠️ Core Skills Demonstrated
Across the projects in this directory, the following core data analytics competencies are showcased:
* **Data Modeling:** Designing robust Star Schema and Snowflake Schema architectures to optimize cross-filtering performance.
* **Advanced DAX:** Implementing complex measures, including Time-Intelligence calculations (YoY, MoM growth), dynamic conditional formatting rules, and KPI KPI blocks.
* **UI/UX Design for Analytics:** Applying strict corporate design principles, visual hierarchy (Z-pattern layout), cohesive color grading, and app-like navigation controls (e.g., localized reset bookmarks).
* **Data Cleansing & ETL:** Normalizing inconsistent display scales, filtering system anomalies/blank fields, and adjusting text truncation issues using Power Query.
